```bash

```

```bash
cd $HOME #Do not write this if you are using c9 or not root accounts
git clone https://github.com/ArchonTeam/supreme-robot
cd Self-Bot
./launch.sh install
./launch.sh # Will ask you for a phone number & confirmation code.
```
